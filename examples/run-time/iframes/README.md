# Iframe Implementation

## Requirements

* Docker

## Technologies

* Docker
* React
* Vue
* NodeJS
  * Express
  * Compression

## Instructions

### Build

```bash
docker-compose build
```

### Run
```bash
docker-compose up
```

### Stop
``` bash
docker-compose down
```

## Access

### Core Application

```bash
  http://localhost:3000
```

### Child Application

```bash
  http://localhost:3001
```
